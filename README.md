# Validation Module for Python

A reusable Python module designed to simplify input validation and common data-handling tasks.

This project provides a collection of well-structured helper functions for validating numbers, text, ranges, and user responses, while also supporting utility operations such as duplicate removal.

## Features

- Validate positive integers
- Validate numeric input
- Check whether a number falls within a specific range
- Ensure text input is not empty
- Validate names containing only alphabetic characters
- Handle yes/no responses consistently
- Remove duplicate items from lists
- Convert repeated user confirmations into boolean values

## Purpose

The goal of this project is to promote clean, reusable, and maintainable code by centralizing validation logic into a single module.

By separating validation from application logic, developers can improve code readability, reduce repetition, and build more reliable programs.

## Technologies Used

- Python 3
- Type Hints
- Exception Handling
- Modular Programming
